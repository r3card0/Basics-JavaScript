# Operadores

Tipos de operadores:

| Operador    |       símbolo(s)      |
|-------------|-----------------------|
| Aritméticos | +, -, /, //, *        |
| Asignación  | =, !=                 |
| Comparación | ==, ===, <, >, <=, >= |
| Lógicos     | && (and), II (or), not|
| variables   | ++, +=                |
| Otros       | !false, !true         |
